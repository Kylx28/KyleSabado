Classes and Objects

An object is a data type that has its own attributes and methods, and is the basis of Object-Oriented Programming.

A class is a Java file created by the user that describes the attributes and behaviors of objects. It is the framework from which objects are created.

Implemetation

How To Implement a Class:

1. You need to give your class a name and create a java file without main. For example, a car class would be.

         Car.java

2. Next, you need to create private instance variables (PIVs) at the top of your class. These PIVs are the attributes of the object this class describes.

         String manufacturer;
         String type;
         String name;
         String colour;

3. Then, you need to create a constructor for your class. This determines which parameters are passed when creating an object.
    You can also have other constructors that take a different number of parameters or have default values.

         public Car(String manufacturer, String type, String name, String colour){
             this.manufacturer = manufacturer;
             this.type = type;
             this.name = name;
             this.colour = colour;
         {

4. You also need to create getter and setter methods for each PIV. These will return each PIV, and allow the user to modify their values.

         public String getName(){
             return name;
         }

         public void setName(String name){
             this.name = name;
         }

5. Setters and getters are the standard methods, but you are able to create any number of unique behavior methods as you want.
    These behavior methods are what give objects their function. For example, here's a method that will compare
    the names of two car objects and return true if they are the same.

         public boolean sameName(Car car2){
             return name.equals(car2.getName);
         }

6. Finally, each class needs a way of printing the object. You typically want to print the PIVs of each object.

         public String toString(){
             return "Manufacturer: " + manufacturer + "\n"
             return "Type: " + type + "\n"
             return "Name: " + name + "\n"
             return "Colour: " + colour;
         }

How To Create Objects:

1. Object are typically created in driver clients. When creating an object, you must give it a name and instantiate it using the "new" keyword.

         Car pista = new Car(Ferrari, Sports Car, 488 Pista, Red);
         Car highlander = new Car(Toyota, SUV, Highlander, Silver);

2. You can use the methods you created on the newly created object. These methods are called by saying object.function(), and filling the brackets with the required parameters.

         pista.getName();
         pista.getColour();
         pista.sameName(highlander);

         Output:
         488 Pista
         Red
         False
