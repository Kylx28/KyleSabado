Recursion

Recursion is a way of solving a problem where the solution depends on the solution of a smaller variation of the same problem.
In programming, this is often done by writing a function that calls itself until it reaches a base case in order to stop the loop.

Implemetation

How To Implement Recursion:

1. Recursion is used on particular types of computational problems. One of the most basic examples is computing factorials.
   In order to avoid making an infinite loop, you need to establish a base case where the loop ends.
   When computing a factorial, the last number is always 1. Thus, the base case is when the input is 1.

2. Recursion involves calling the function inside of itself. In this case, when the function takes an input n, the factorial is computed
   using the formula n(n-1)(n-2)(n-3)...3 x 2 x 1. So in the function, n * function(n-1) should be returned. When n = 1, the base case takes precedent.

Recursive Method for Computing factorials:
         public int factorial(int n){
             if(n == 1){
                 return 1;
             }
             n * factorial(n-1);
         }

Example of Recursion

This method returns the number of 7s in an string of digits.
