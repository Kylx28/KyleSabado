Arrays and ArrayLists

An array is a data structure that holds a fixed number of elements of the same type. Arrays can store primitive data types or objects.

An ArrayList is a data structure that can hold a variable number of elements of the same type. ArrayLists can only hold references to objects found in different locations in memory. Thus, primitive data types are autoboxed into ArrayLists.

Implemetation

How To Implement an Array:

1. First, you need to define what variable type you want to store, followed by a pair of square brackets and the array name.

         String[] Array;

2. Now, you can declare a literal array, where you type out the values inside curly braces separated by commas.

         String[] Array = {"Ferrari", "Mercedes", "Red Bull", "McLaren"}

3. Or, you can instatiate an array with a fixed size using the "new" keyword followed by the datatype and the desired size within square brackets.

         String[] Array = new String[4]

4. If you choose to instatiate like this, you can add values to the array by assigning elements to each index.

         Array[0] = "Ferrari"
         Array[1] = "Mercedes"
         Array[2] = "Red Bull"
         Array[3] = "McLaren"




How To Implement an ArrayList:

1. First, you must import ArrayList from the java.util library

         import java.util.ArrayList;

2. Then, you must create an ArrayList object and specify the data type you want to store inside the ArrayList

         ArrayList<String> array2 = new ArrayList<String>(4)

3. After instantiating the ArrayList, you can add any number of elements to the list using the .add function.

         array2.add("Ferrari")
         array2.add("Mercedes")
         array2.add("Red Bull")
         array2.add("McLaren")


Array and ArrayList Methods

array.length

.length is the most important array method in Java. It returns the number of elements in an array as an integer.

array2.add()

.add() is an overloaded method, meaning it can take either 1 or 2 parameters.
You can add an element at a particular index, which will shift all the elements to the right of the specified index.

         Array2.add(1, "Aston Martin")

Or, you can add an element to the end of the list without specifying the index.

         Array2.add("Aston Martin")

array2.get()

.get() is used to return an element at a specific index.

array2.indexOf()

.indexOf returns the index of the first occurance of an element in the list. It returns -1 if the index is not found.

array2.set()

.set() replaces an element at a particular index with another element.

array2.size()

.size() is synonymous with .length for arrays. It returns the number of elements in the list.

Printing Arrays and ArrayLists

Simply printing an array using System.out.println will print out the actual address of the array in memory, which looks like a random assortment of characters.
Printing an ArrayList in the same way will return each of the array elements within a set of square brackets, but this is not a particularly good way to print the contents of the array.

The proper way to print elements of an array or an ArrayList is to print the elements individually.

         System.out.println(array[0])
         System.out.println(array2.get(1))

         Output:
         Ferrari
         Mercedes

The best way to print all the elements of an array or ArrayList is to iterate through each index using a for loop.

         for(int i = 0; i < array.length; i++){
             System.out.println(array[i]);
         }

         Output:
         Ferrari
         Mercedes
         Red Bull
         McLaren

         for(int i = 0; i < array2.size(); i++){
             System.out.println(array2.get(i));
         }

         Output:
         Ferrari
         Mercedes
         Red Bull
         McLaren
