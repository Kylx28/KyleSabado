Arrays and ArrayLists

An array is a data structure that holds a fixed number of elements of the same type. Arrays can store primitive data types or objects.

An ArrayList is a data structure that can hold a variable number of elements of the same type. ArrayLists can only hold references to objects found in different locations in memory. Thus, primitive data types are autoboxed into ArrayLists.

Implemetation

How To Implement an Array:

1. First, you need to define what variable type you want to store, followed by a pair of square brackets and the array name.

         String[] arr;

2. Now, you can declare a literal array, where you type out the values inside curly braces separated by commas.

         String[] arr = {"Ferrari", "Mercedes", "Red Bull", "McLaren"}

3. Or, you can instatiate an array with a fixed size using the "new" keyword followed by the datatype and the desired size within square brackets.

         String[] arr = new String[4]

4. If you choose to instatiate like this, you can add values to the array by assigning elements to each index.

         String[0] = "Ferrari"
         String[1] = "Mercedes"
         String[2] = "Red Bull"
         String[3] = "McLaren"




How To Implement an ArrayList:

1. First, you must import ArrayList from the java.util library

         import java.util.ArrayList;

2. Then, you must create an ArrayList object and specify the data type you want to store inside the ArrayList

         ArrayList<String> = new ArrayList<String>(4)

3. After instantiating the ArrayList, you can add any number of elements to the list using the .add function.

         array.add("Ferrari")
         array.add("Mercedes")
         array.add("Red Bull")
         array.add("McLaren")


Printing Arrays and ArrayLists

Simply printing an array using System.out.println will print out the actual address of the array in memory, which looks like a random assortment of characters.
Printing an ArrayList in the same way will return each of the array elements within a set of square brackets, but this is not a particularly good way to print the contents of the array.
